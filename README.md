# term-ext

Скрипт для разворачивания терминального сетапа одной командой (Linux + macOS).

## Что устанавливается

- **starship** — кастомный prompt
- **fzf** — fuzzy search
- **fzf-tab** — автодополнение через fzf
- **zoxide** — умный `cd`
- **eza** — красивый `ls`
- **bat** — красивый `cat`
- **broot** — файловый менеджер
- **lazydocker** — TUI для Docker
- **lazygit** — TUI для Git

## Скрипты

| Скрипт | Когда использовать |
|--------|-------------------|
| `install-terminal.sh` | Свой сервер / Mac, ставит системно (sudo / brew) |
| `install-terminal-user.sh` | Общий сервер, без sudo, только для текущего пользователя |
| `install-terminal-local.sh` | Личная машина: tmux, claude, ralphex |

## Установка

### Свой сервер или Mac

```bash
curl -fsSL https://raw.githubusercontent.com/kar43lov/term-ext/main/install-terminal.sh -o /tmp/term-ext.sh && bash /tmp/term-ext.sh && exec zsh
```

### Общий сервер (без sudo, только для меня)

```bash
curl -fsSL https://raw.githubusercontent.com/kar43lov/term-ext/main/install-terminal-user.sh -o /tmp/term-ext-user.sh && bash /tmp/term-ext-user.sh && exec zsh
```

### Личная машина (tmux + claude + ralphex)

Ставится после основного скрипта:

```bash
curl -fsSL https://raw.githubusercontent.com/kar43lov/term-ext/main/install-terminal-local.sh -o /tmp/term-ext-local.sh && bash /tmp/term-ext-local.sh && exec zsh
```

> На macOS `curl | bash` не работает из-за brew, который перехватывает stdin. Скачивай в файл.

### Переустановка / обновление

Чтобы получить новые команды и фиксы — запусти ту же команду установки ещё раз. Инструменты, которые уже стоят, пропустятся, а `~/.zshrc.term-ext` обновится.

Скрипты пишут управляемые конфиги в `~/.zshrc.term-ext` и `~/.zshrc.term-ext-local`, а в `~/.zshrc` добавляется `source`. Повторный запуск обновляет только управляемые файлы — пользовательские добавки в `~/.zshrc` не затираются.

После обновления:

```bash
exec zsh
```

## Требования

- Linux (x86_64, aarch64) или macOS (Apple Silicon, Intel)
- macOS: Homebrew
- `install-terminal.sh` на Linux: sudo-доступ
- `install-terminal-user.sh` на Linux: только git и curl

## Шпаргалка по командам

> Прямо в терминале:
> - `help-ext` — все команды основного сетапа
> - `help-local` — команды локальных расширений (tmux, claude, ralphex)

### zoxide — умная навигация по папкам

| Команда | Что делает |
|---------|-----------|
| `z foo` | Перейти в папку, содержащую `foo` в пути (из истории посещений) |
| `z foo bar` | Перейти в папку, содержащую `foo` и `bar` |
| `zi` | Интерактивный выбор папки из истории (через fzf) |
| `z -` | Вернуться в предыдущую папку |

zoxide запоминает папки, в которых ты бывал. Чем чаще посещаешь — тем выше приоритет.

### fzf — нечёткий поиск

| Сочетание | Что делает |
|-----------|-----------|
| `Ctrl+R` | Поиск по истории команд |
| `Ctrl+T` | Поиск файлов в текущей папке |
| `Alt+C` | Поиск и переход в папку |

В любом поиске fzf: начни вводить текст для фильтрации, `Enter` — выбрать, `Esc` — отмена.

### fzf-tab — автодополнение

| Действие | Что делает |
|----------|-----------|
| `cd <Tab>` | Выпадающий список папок с превью дерева файлов |
| `git checkout <Tab>` | Выбор ветки через fzf |
| любая команда + `<Tab>` | Автодополнение аргументов через fzf |

### Стрелки — поиск по истории

| Клавиша | Что делает |
|---------|-----------|
| `↑` | Предыдущая команда, начинающаяся с уже введённого текста |
| `↓` | Следующая команда, начинающаяся с уже введённого текста |

Пример: набери `git` и нажми `↑` — будет листать только команды, начинающиеся с `git`.

### eza — красивый ls

| Команда | Что делает |
|---------|-----------|
| `ls` | Список файлов (алиас на `eza`) |
| `ll` | Список с подробностями (`eza -l`) |
| `la` | Все файлы, включая скрытые (`eza -la`) |
| `lt` | Дерево файлов на 2 уровня (`eza --tree --level=2`) |

### bat — красивый cat

| Команда | Что делает |
|---------|-----------|
| `cat файл` | Показать файл с подсветкой синтаксиса (алиас на `bat`) |

### broot — файловый менеджер

| Команда | Что делает |
|---------|-----------|
| `br` | Открыть файловый менеджер в текущей папке |
| `br /path` | Открыть файловый менеджер в указанной папке |

Внутри broot:
- Начни вводить текст — фильтрация файлов
- `Enter` — перейти в папку / открыть файл
- `Ctrl+E` — открыть файл в редакторе
- `Ctrl+T` — открыть терминал в текущей папке
- `Esc` / `Ctrl+Q` — выйти

### Утилиты

| Команда | Что делает |
|---------|-----------|
| `mkcd имя` | Создать папку и сразу перейти в неё |

### Git — lazygit

| Команда | Что делает |
|---------|-----------|
| `lg` | Открыть lazygit (TUI для Git) |

Внутри lazygit (основное):
- стрелки / `j`/`k` — навигация
- `Space` — stage / unstage файла
- `a` — stage / unstage всего
- `c` — commit (откроется редактор для сообщения)
- `P` — push, `p` — pull
- `s` — stash, `S` — pop stash
- `b` — список веток, в нём `Space` — checkout, `n` — новая ветка
- `?` — справка по контексту, `q` — выйти

### Docker — управление контейнерами

Все команды без аргумента открывают fzf-выбор контейнера. С аргументом — работают по имени напрямую.

| Команда | Что делает |
|---------|-----------|
| `ld` | Открыть lazydocker (TUI для Docker) |
| `dps [фильтр]` | Список контейнеров (`dps mup` — только с `mup` в имени) |
| `dr [имя]` | Restart контейнера |
| `ds [имя]` | Stop контейнера |
| `dst [имя]` | Start контейнера |
| `drm [имя]` | Удалить контейнер (`rm -f`) |
| `dlogs [имя]` | Логи контейнера в реальном времени |

Автоопределение sudo: если `docker ps` работает без sudo — команды без sudo, иначе с sudo (включая `ld` — резолвит абсолютный путь к `lazydocker`, чтобы PATH не сбился).

Doznanie-команды (`dozrun`, `dozrun-kafka`, `dozrun-gost` и т.д.) вынесены в отдельный репозиторий [term-ext-doz](https://github.com/kar43lov/term-ext-doz).

### Локальные расширения (install-terminal-local.sh)

#### tmux

| Команда | Что делает |
|---------|-----------|
| `ta имя` | Подключиться к tmux-сессии |
| `tls` / `tl` | Список tmux-сессий |
| `tk имя` | Удалить tmux-сессию |
| `ts` | Интерактивный выбор сессии через fzf (Ctrl-D — удалить) |

#### Claude

| Команда | Что делает |
|---------|-----------|
| `cl` | Алиас на `claude` |
| `tc` | Запуск Claude в tmux-сессии (если уже в tmux — запускает напрямую) |

#### Ralphex

| Команда | Что делает |
|---------|-----------|
| `ralph` | Запуск ralphex напрямую |
| `ralphex` | Запуск ralphex в tmux-сессии |

### starship — prompt

Prompt автоматически показывает:
- текущую папку
- ветку git (`🌱 main`)
- статус git (изменённые файлы)
- время выполнения долгих команд (`⏱ 5s`)
- `➜` при успехе, `✗` при ошибке предыдущей команды

### Шпаргалки в терминале

| Команда | Что показывает |
|---------|----------------|
| `help-ext` | Все команды основного сетапа (zoxide, fzf, eza, bat, broot, mkcd, Docker) |
| `help-local` | Команды локальных расширений (tmux, claude, ralphex) — только если стоит `install-terminal-local.sh` |

Шпаргалки автоматически скрывают разделы под отсутствующие инструменты — например, блок Docker не показывается, если `docker` не установлен.
